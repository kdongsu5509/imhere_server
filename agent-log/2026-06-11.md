### Work Done
- Removed the duplicate `@PostMapping` form-parameter controller method from `TermsAdminPageController.kt`.
- Updated `SecurityConfig.kt` to split the security filter chains into:
  - `adminApiFilterChain` (Order 1, `/api/admin/**` using sessions, stateless-like 401 handling, CSRF disabled).
  - `adminWebFilterChain` (Order 2, `/admin/**` using sessions, login redirect, CSRF enabled).
  - `apiFilterChain` (Order 3, `/api/**` using stateless JWT).
- Updated `terms.html` to handle registration via an asynchronous fetch POST request to `/api/admin/terms`, formatting the date input, handling success notifications via sessionStorage, and reloading the page dynamically.
- Added mock beans for `ForceLogoutUseCase` to `UserAdminControllerWebMvcTest.kt` and `UserCommandControllerWebMvcTest.kt` to fix pre-existing UnsatisfiedDependencyExceptions in the user domain test suite.
- Ran the test suite verifying that all 541 tests pass cleanly.
