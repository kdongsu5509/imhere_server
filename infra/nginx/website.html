주요 기능

알아서 보내는 출발·도착

매번 전화하거나 위치를 물어볼 필요 없어요.
지정해 둔 장소의 경계를 오갈 때
자동으로 출발과 도착을 알려줘요.

안전한 친구끼리만

서로 수락한 친구하고만
안전하게 공유하니까,
모르는 사람에게 내 위치가
노출될 걱정이 전혀 없어요.

실시간 경로는 비공개

실시간 이동 경로는
어디에도 수집되지 않아요.
오직 사전에 설정한
출발과 도착 시점만 깔끔하게 전달합니다.

개인정보 보호

내 위치 정보는
내 폰에만 저장되니까

실시간 위치는 서버에 절대 저장되지 않아요.
모든 위치 계산은 오직 내 휴대폰 기기 안에서만(On-Device) 안전하게 처리하고,
친구에게는 오직 출발·도착 알림만 전송합니다.
